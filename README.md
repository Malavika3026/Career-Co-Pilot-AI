# Career Co-Pilot AI

## Overview
Career Co-Pilot AI is a Streamlit-based application that analyzes user profiles and job data to suggest suitable career paths and identify skill gaps.

## Features
- Role recommendation
- Skill gap analysis
- Career roadmap generation
- Interactive Streamlit UI

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn

## Dataset
Job and company datasets used for career and skill analysis.

## How to Run the Project

1. Clone the repository:
git clone https://github.com/Malavika3026/Career-Co-Pilot-AI.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the Streamlit app:
streamlit run app/streamlit_app.py
